## Requirements

* PHP 8.0 or latest

## How to run

```bash
$ cp .env.example .env # create your database first, and then configure your database connection
$ composer install
$ npm install
$ npm run dev # optional only if any changes you make in resource/js
$ php artisan migrate
$ php artisan serve
```

## Compile Assets

```bash
$ npm run dev
```

or

```bash
$ npm run watch
```
