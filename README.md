## Requirements

* PHP 8.0 or latest
* Node 16+ or latest

## How to run

```bash
$ cp .env.example .env # create your database first, and then configure your database connection
$ composer install
$ php artisan migrate --seed
$ php artisan storage:link
$ php artisan serve
$ npm install
$ npm run dev # compiling asset for development
```
